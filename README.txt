You are building an address book application.  Build a page that displays all of the users in the address book.

The API to retrieve all users, as well as their metadata:
* GET /users
* Base URL: jsonplaceholder.typicode.com
* Scheme: https

Design the page as nice as you can.

Bonus Features
* Be able to sort by name or username
* Support two display modes (list & grid)
* User's address is hidden by default, but can be displayed on click (per user)
